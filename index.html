About

I’m a Senior Technical Writer and Knowledge Architect with experience creating documentation for SaaS platforms, developer portals, enterprise software, and IT products.

My work spans developer documentation, API content, help center articles, release notes, onboarding materials, knowledge bases, and documentation architecture.
